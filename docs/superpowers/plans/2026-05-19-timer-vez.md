# Timer de 30 Segundos por Vez — Plano de Implementação

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Adicionar um timer de 30s por vez visível a todos os clientes (anel SVG circular), com fallback de 32s no servidor e bloqueio do botão "Passar Vez" após o jogador enviar resposta.

**Architecture:** O servidor adiciona `turnStartedAt: Date.now()` nos eventos que trocam a vez. Cada cliente calcula o tempo restante via `setInterval(100ms)` a partir desse timestamp. Um `setTimeout(32s)` em `GerenciadorJogo` auto-passa a vez no servidor caso o cliente não responda, disparando o broadcast via callback registrado em `ws.ts`.

**Tech Stack:** Bun + TypeScript (backend), Next.js 16 App Router + React 19 + SVG inline (frontend).

```tsx
function CountdownRing({ timeLeft, visible }: { timeLeft: number; visible: boolean }) {
  if (!visible) return null;
  const r = 18;
  const strokeWidth = 4;
  const size = 46;
  const cx = size / 2;
  const circumference = 2 * Math.PI * r;
  const dashoffset = circumference * (1 - Math.min(timeLeft, 30) / 30);
  const isUrgent = timeLeft <= 10;
  const color = isUrgent ? '#EF4444' : '#C4B5FD';

  return (
    <div style={{ position: 'relative', width: size, height: size, flexShrink: 0 }}>
      <svg width={size} height={size}>
        <circle cx={cx} cy={cx} r={r} fill="none" stroke="rgba(255,255,255,0.1)" strokeWidth={strokeWidth} />
        <circle
          cx={cx} cy={cx} r={r}
          fill="none"
          stroke={color}
          strokeWidth={strokeWidth}
          strokeDasharray={circumference}
          strokeDashoffset={dashoffset}
          strokeLinecap="round"
          transform={`rotate(-90 ${cx} ${cx})`}
          style={{ transition: 'stroke-dashoffset 0.1s linear, stroke 0.5s ease' }}
        />
      </svg>
      <span
        className={isUrgent ? 'animate-pulse' : ''}
        style={{
          position: 'absolute', inset: 0,
          display: 'flex', alignItems: 'center', justifyContent: 'center',
          fontSize: '0.7rem', fontWeight: 'bold', color,
        }}>
        {Math.ceil(timeLeft)}
      </span>
    </div>
  );
}
```

> `strokeDashoffset = 0` quando `timeLeft = 30` (círculo cheio). `strokeDashoffset = circumference` quando `timeLeft = 0` (círculo vazio). `rotate(-90)` faz o arco começar do topo.
